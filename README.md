# Document Categorizer

Автоматическая сортировка тысяч заметок с помощью ИИ.

---

## Для кого это

У вас накопились тысячи заметок в Evernote, Notion или где-то ещё. Они лежат кучей — без структуры, без тегов. Найти что-то невозможно. Разобрать вручную — недели работы.

Этот инструмент читает каждую заметку, понимает о чём она, и раскладывает по папкам. Автоматически. За пару часов вместо недель.

---

## Что нужно для начала

### 1. Obsidian (бесплатно)

[Obsidian](https://obsidian.md/) — редактор заметок, в котором вы будете работать с результатом.

**Почему именно он:**
- Все заметки хранятся у вас на компьютере (не в облаке)
- Клик по ссылке → открывается связанная заметка
- Клик по тегу → все заметки с этим тегом
- Красивый граф связей между заметками

**Как установить:** Скачайте с [obsidian.md](https://obsidian.md/) и установите. Это займёт 2 минуты.

> Технически можно работать и без Obsidian — результат это обычные текстовые файлы. Но с ним удобнее.

### 2. Claude Code или другой AI-ассистент

[Claude Code](https://claude.com/claude-code) — это ИИ-помощник который умеет выполнять команды на вашем компьютере. Он сам установит всё что нужно и запустит сортировку.

**Альтернативы:** [Cursor](https://cursor.com/), [Windsurf](https://codeium.com/windsurf), [Cline](https://github.com/cline/cline)

**Почему через ассистента:** Вам не нужно знать командную строку, Python или git. Просто говорите что хотите — ассистент делает.

### 3. Anthropic API ключ (~$3 за 1000 заметок)

Сортировка работает через Claude API. Нужен ключ доступа.

**Как получить:**
1. Зайдите на [console.anthropic.com](https://console.anthropic.com/)
2. Зарегистрируйтесь
3. Пополните баланс (минимум $5)
4. Создайте API ключ в разделе API Keys

## Как запустить (простой способ)

Откройте Claude Code (или другой AI-ассистент) и скопируйте этот промпт:

```
Помоги мне настроить и запустить document-categoriser для сортировки моих заметок.

Репозиторий: https://github.com/aturilin/document-categoriser

Работай в диалоговом режиме — задавай мне вопросы по одному и жди ответа:

1. УСТАНОВКА
   - Склонируй репозиторий
   - Установи Python-зависимости
   - Спроси у меня Anthropic API ключ и создай .env файл

2. НАСТРОЙКА
   - Спроси где лежат мои заметки (путь к папке с .md файлами)
   - Спроси на каком языке заметки (русский/английский/смешанный)
   - Покажи текущие категории в categorize.py и спроси нужно ли их изменить
   - Если да — помоги настроить категории под мои нужды

3. КАЛИБРОВКА (важно!)
   - Запусти тест на 10 случайных заметках: python3 categorize.py --dry-run --limit 10
   - Покажи мне результаты: какие заметки куда попали
   - Спроси доволен ли я результатом
   - Если нет — помоги скорректировать категории или промпт
   - Повтори калибровку пока результат не устроит

4. ЗАПУСК
   - После успешной калибровки спроси готов ли я к полному запуску
   - Запусти на всей базе: python3 categorize.py
   - Показывай прогресс
   - После завершения запусти python3 generate_moc.py для навигации

5. РЕЗУЛЬТАТ
   - Покажи статистику: сколько заметок куда попало
   - Объясни как открыть результат в Obsidian

Начни с первого шага. Жди моих ответов перед следующим действием.
```

**Что будет происходить:**

1. Ассистент спросит ваш API ключ
2. Спросит где лежат заметки
3. Покажет категории и спросит подходят ли они
4. Запустит тест на 10 заметках и покажет результат
5. Спросит всё ли ок или нужно подкрутить
6. После калибровки запустит на всей базе
7. Построит навигацию и покажет как открыть в Obsidian

Весь процесс в диалоге — вы контролируете каждый шаг.

## Что получится

Ваши заметки будут разложены по папкам:

```
output/
├── areas/                  # Текущие сферы жизни
│   ├── health/            # здоровье, фитнес
│   ├── finance/           # деньги, счета
│   ├── career/            # работа, карьера
│   └── family/            # семья, дети
├── resources/             # Справочные материалы
│   ├── programming/       # код, технологии
│   ├── business/          # бизнес, стратегия
│   └── personal-dev/      # книги, саморазвитие
├── archive/               # Неактуальное
│   └── outdated/          # старое, завершённое
└── _MOC/                  # Навигационные хабы
    ├── health.md          # Все заметки про здоровье
    ├── finance.md         # Все заметки про финансы
    └── ...
```

Каждая заметка получит теги и краткое описание в начале файла.

## Методологии

### PARA — как организованы папки

[PARA](https://fortelabs.com/blog/para/) — система от Tiago Forte (книга [Building a Second Brain](https://www.buildingasecondbrain.com/)):

| Папка | Что там | Пример |
|-------|---------|--------|
| **Areas** | То чем занимаетесь постоянно | здоровье, финансы, семья |
| **Resources** | Полезное на будущее | книги, статьи, туториалы |
| **Archive** | Неактуальное | старые проекты, устаревшее |

### MOC — как устроена навигация

[Maps of Content](https://notes.linkingyourthinking.com/Cards/MOCs+Overview) — это страницы-оглавления.

Например `_MOC/health.md` содержит ссылки на ВСЕ ваши заметки про здоровье, сгруппированные по темам. Открыли хаб — видите всё что есть по теме.

## Ручная установка (для продвинутых)

Если хотите делать всё сами:

```bash
# 1. Скачать
git clone https://github.com/aturilin/document-categoriser.git
cd document-categoriser

# 2. Установить зависимости
pip install -r requirements.txt

# 3. Добавить API ключ
echo 'ANTHROPIC_API_KEY=sk-ant-ваш-ключ' > .env

# 4. Положить заметки в input/
cp /путь/к/заметкам/*.md input/

# 5. Запустить (сначала тест на 5 файлах)
python3 categorize.py --dry-run --limit 5

# 6. Запустить по-настоящему
python3 categorize.py

# 7. Построить навигацию
python3 generate_moc.py
```

**Откуда взять заметки в формате .md:**

- **Evernote:** File → Export → ENEX, потом конвертировать через [evernote2md](https://github.com/wormi4ok/evernote2md)
- **Notion:** Export → Markdown
- **Obsidian:** Просто скопировать папку vault

## Вопросы

**Процесс прервался — что делать?**
```
python3 categorize.py --resume
```
Продолжит с того места где остановился.

**Хочу другие категории**
Отредактируйте файл `categorize.py`, секция `CATEGORIES`.

**Можно ли без Obsidian?**
Да. Результат — обычные `.md` файлы. Откроются в любом текстовом редакторе.
